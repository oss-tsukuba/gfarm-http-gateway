# gfarm-http-gateway

HTTP gateway for Gfarm

## Setup

- ./setup.sh

## Start server

### Start for clients on localhost (127.0.0.1)

- ./bin/start.sh

### Start for clients on any hosts

- ./bin/start.sh 0.0.0.0:8000

### for developer

- ./bin/start-dev.sh
